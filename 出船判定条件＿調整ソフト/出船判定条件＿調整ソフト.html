📋 使い方

  1. 下の各ファクターの数値を変更する（変更されたフィールドはオレンジ色で表示）
  2. 「💾 criteria.json を出力」ボタンをクリックして保存
  3. GitHub リポジトリの 出船判定条件＿調整ソフト/criteria.json を出力したファイルで上書きアップロード
  4. GitHub Pages 経由で出船判断システム（index.html）が自動的に新しい基準を参照する
