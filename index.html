室戸沖 出船判断システム

MUROTO OFFSHORE FORECAST SYSTEM

室戸沖（高知県）への遊漁船出船可否を、気象・海況データから自動判定するシステムです。 Open-Meteo の Marine API と Weather API から波高・風速を取得し、5日分のリアルタイム予報を提供します。

🌊 出船判断を確認する
📍 LAT 33.2 / LON 134.2 ⏱ 6時・9時・12時 × 5日間 🔄 リアルタイム取得
🌊
海況データ自動取得

Open-Meteo Marine API から波高データを、Weather API から風速・風向・降水量データをリアルタイムに取得。認証不要・完全無料で利用できます。

⚖️
独自スコアリング

波高だけでなく風速のトレンド（増加・減少）を加味した補正スコアを算出。同じ波高でも風が強まっている場合は危険度を高く評価します。

📊
グラフ＆CSV出力

波高・風速の推移を折れ線グラフで可視化。判定結果はCSVでダウンロード可能。予約管理システムへの連携データとして活用できます。

判定基準 — SCORING CRITERIA
◎ 釣り日和 波高 ≤ 1.0m
○ 出船可能 波高 ≤ 1.3m
△ 条件付き 波高 ≤ 1.7m
× 出船不可 波高 > 1.7m

※ 風速トレンド補正あり — 前時刻との差が 0.5m/s 以上の場合、増加時はスコアを1段階悪化、減少時は1段階改善します。

データソース — DATA SOURCES
API 取得データ エンドポイント
Open-Meteo Marine API 波高（合成波高） marine-api.open-meteo.com/v1/marine
Open-Meteo Weather API 風速・風向・降水量 api.open-meteo.com/v1/forecast
